:q --> exit only.
:q! -> quit without saving.
:x  --> save and exit
to view the data inside the file : cat filename

to copy from one file to another : cp file1 file2
rename file : mv file1 file2
delete file: rm filename
delete folder: rmdir folder
delete folder including files in it: rm -r folder
